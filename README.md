## 🚀 Overview

**GradeForge** is a sleek and powerful GPA calculator that helps students compute their **semester GPA and cumulative CGPA** with precision.

It supports:

* 📚 Multiple semesters
* 🧮 Credit-based GPA calculation
* 📊 Real-time analytics & visualization
* 💾 Local storage persistence

---

## ✨ Features

### 📌 Core Functionalities

* Add unlimited **subjects & semesters**
* Automatic **GPA calculation**
* **CGPA tracking** across semesters
* Grade-to-point conversion system

### 📊 Smart UI Components

* Dynamic GPA meter (arc visualization)
* Grade badge with color indicators
* Semester history tracking
* Clean dashboard-style layout

### ⚡ Advanced Features

* 💾 Data saved using **LocalStorage**
* 📥 Export GPA summary as `.txt`
* 🔄 Reset & clear options
* 📱 Fully responsive design

---

## 🧠 GPA Formula Used

[
GPA = \frac{\sum (Credits \times GradePoints)}{\sum Credits}
]
